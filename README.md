# Beast Mode 2-A-Day Shred

**12-Week Beast Mode Training Program** — High-intensity 2-a-day fitness app built with React.

An interactive, visually aggressive workout planner designed for serious lifters chasing **maximum fat loss and muscle retention**.

![Beast Mode Preview](https://via.placeholder.com/800x420/0a0a0a/FF4500?text=BEAST+MODE+2-A-DAY+SHRED)

## ✨ Features

- 3 Progressive Phases (Ignition → Inferno → Shred)
- Daily 2-A-Day Training (Morning HIIT/Cardio + Evening Lifting)
- Interactive Phase & Day Selector
- Built-in YouTube exercise demo videos (click "▶ WATCH")
- Supersets, Giant Sets & Circuits clearly highlighted
- Realistic weekly calorie burn estimates
- Brutal dark aesthetic with strong visual hierarchy
- Active Recovery Sunday

### Weekly Split
- **Monday** – Chest + Triceps + Abs  
- **Tuesday** – Back + Biceps + Abs  
- **Wednesday** – Full Body HIIT + Core  
- **Thursday** – Legs + Glutes + Abs  
- **Friday** – Shoulders + Traps + Abs  
- **Saturday** – Beast Mode Full Body Circuit  
- **Sunday** – Active Recovery / Mobility

## 🛠 Tech Stack

- React (Functional Components + Hooks)
- JavaScript / JSX
- Pure inline styling (no external CSS files)
- YouTube iframe embeds
